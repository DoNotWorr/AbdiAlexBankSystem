Git:
1. git branch //visar alla brancher
2. git checkout -b <namn> //skapar en ny branch som heter <namn>
3. git checkout <name> //ändrar vilken branch man ställer sig på

git commit
git status //visar min commit (mina ändringar)



När man är klar med pull request och merge:
git checkout master
git pull
//gör ett test att allt funkar
